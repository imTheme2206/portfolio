# Curate Project Playground media

Type: HITL

## What to build

Choose the strongest publishable media for each side project and approve the concise story each entry tells. Prefer existing screenshots and lightweight motion treatments; identify only the few interactions that justify recording additional video or creating diagrams.

## Acceptance criteria

- [ ] Every project has an approved representative image and meaningful alternative text.
- [ ] Project summaries state purpose and distinctive behavior before listing technologies.
- [ ] Live, source-only, inactive, and in-progress destinations are labelled accurately.
- [ ] Any additional video or diagram work is explicitly prioritized and scoped.
- [ ] No project is positioned as the conceptual backbone of the full portfolio.
