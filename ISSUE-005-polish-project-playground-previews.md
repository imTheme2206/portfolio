# Polish Project Playground previews

Type: AFK

## What to build

Populate the reusable Project Playground with the approved media and tune each preview so screenshots, subtle pans, scan-light details, and any optional recordings feel intentional within the same theatre. Preserve a consistent interaction grammar while allowing each project to retain its visual character.

## Acceptance criteria

- [ ] Approved media is optimized, correctly sized, and displayed without unintended cropping.
- [ ] Preview treatments use transforms and opacity without layout thrashing.
- [ ] Optional recordings pause when off-screen and have a static fallback.
- [ ] Project changes remain smooth during fast scrolling and direct selection.
- [ ] All project destinations and statuses match the approved content inventory.
